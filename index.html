SELAMAT DATANG DI PEMBELAJARAN DASAR HTML

HTML (HyperText Markup Language) adalah bahasa markah yang digunakan untuk membuat struktur dan mengatur konten pada halaman web. Ini adalah elemen dasar yang membentuk kerangka kerja halaman web dan memberikan instruksi kepada peramban (browser) tentang bagaimana tampilan dan interaksi dengan elemen-elemen tersebut. HTML menggunakan tag untuk mengidentifikasi dan mengelompokkan elemen-elemen dalam dokumen. Setiap tag dimulai dengan tanda < dan diakhiri dengan tanda >. Ada tag pembuka (opening tag) dan tag penutup (closing tag) untuk sebagian besar elemen, meskipun beberapa elemen, seperti gambar, hanya memerlukan tag pembuka. Misalnya, tag 'h1' digunakan untuk membuat judul utama, tag 'p' digunakan untuk menandai paragraf teks, dan tag 'a' digunakan untuk membuat tautan (link) ke halaman lain. Atribut dapat ditambahkan ke dalam tag untuk memberikan informasi tambahan tentang elemen. Contohnya, atribut href dalam tag 'a' digunakan untuk menentukan URL tautan.
